## База ООП
Задача: разработать многоуровневую банковскую систему, включающую модели счетов, систему транзакций, процессы аудита, отчётность, а также элементы безопасности и расширяемости. Все части проекта должны быть реализованы в виде объектно-ориентированных модулей на Python.
## Запуск
Для запуска общей функциональности проекта запустить файл src/main.py
Для тестирования отдельных модулей, соответствующих заданиям из каждого дня, запустить файл tests/file.py
